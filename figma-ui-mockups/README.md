# Salon Admin UI Mockups

This folder contains Figma-style PNG mockups for a salon and beauty retail admin system.

Run the generator:

```powershell
powershell -ExecutionPolicy Bypass -File .\generate-mockups.ps1
```

The V2 visual direction follows the supplied screenshots more closely: white sidebar, orange primary actions, cyan/navy/green summary cards, GHC sales data, compact search/filter controls, and cleaner table/form states.
